# Flagship dense-workbench directive

Date: 2026-04-11
Audience: `chummer6-ui`
Status: injected fleet feedback

## Summary

Avalonia is the flagship desktop route.
It is not allowed to feel like a dashboard wrapped around a form.

For veteran acceptance, the flagship desktop head must read like a dense Chummer workbench:

* real top menu
* immediate toolstrip
* dense center workbench
* stable tab and section rhythm
* contextual inspector
* permanent bottom status strip

## Required changes

* Make a classic dense workbench posture the default for flagship desktop screens.
* Remove or sharply reduce oversized banners, giant section headers, nested cards, decorative chrome, and spacing that lowers visible information density.
* Add release proof for:
  * Chummer5a screenshot comparison
  * first-minute veteran orientation
  * task speed for new/open/import/save/sourcebook/settings/roster/print-export/dice flows
  * no dashboard-first startup or restore detour
* Treat any screen that hides menu/toolstrip/status-strip rhythm behind decorative shells as a flagship regression.

## Release bar

Feature presence is not enough.
The flagship Avalonia head must prove that a Chummer5a veteran can start working immediately without relearning the client.
